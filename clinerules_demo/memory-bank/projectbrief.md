## Project Brief

This project automates login functionality validation for https://www.saucedemo.com using Selenium WebDriver with TestNG test framework in Java 17.  
The objectives are:
- Verify multiple user roles can log in successfully to the application.
- Capture post-login screenshots for audit and debugging.
- Maintain full automation through Maven for builds, dependencies, and test lifecycle.
- Enable maintainable, easily extensible regression automation in a reusable structure.
